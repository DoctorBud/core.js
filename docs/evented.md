## Evented

Use as `import Evented from 'evented'`.

### `new Evented() => Evented`
Creates a new instance of the evented class which is a wrapper for handling events and listening to properties.

#### Methods

##### `.on(events: String, fn: Function)`

##### `.off(events: String, fn: Function)`

##### `.trigger(events: String, ...args: Any)`

##### `.set(property: String, value: Any)`

##### `.listen(property: String, fn: Function)`

##### `.get(property: String)`
